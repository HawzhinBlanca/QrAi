# Quran AI Agents Service

Agents are supervised workflow tools, not religious authorities.

Required agents:

- Recitation Coach
- Tajweed Explainer
- Memorization Planner
- Teacher Copilot
- Curriculum Builder
- Localization Agent
- Support Agent
- Data QA Agent
- Scholar Review Agent

Every learner-facing agent answer must pass the source/review gate in `packages/contracts` before display.
